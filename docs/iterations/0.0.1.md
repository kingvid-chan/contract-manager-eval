# 0.0.1 — MVP 合同管理系统

## 范围
- 登录认证（JWT）+ 演示账号
- 用户管理（admin CRUD, 角色分配）
- 合同 CRUD + 状态流转（草稿→待签→已签→终止/过期）
- 附件上传下载（pdf/word 类型校验, ≤10MB）
- 操作审计日志
- 公网部署到 http://120.24.117.67/projects/contract-manager-eval/

## 业务模型
详见 events e000002 `requirement_defined`

- User: username, password_hash, display_name, email, role(admin/manager/viewer), status
- Contract: contract_no, title, party_a, party_b, amount, status, sign_date, start_date, end_date, description
- Attachment: contract_id FK, filename, original_name, file_type, file_size, storage_path
- AuditLog: user_id FK, action, entity_type, entity_id, detail

## 产品流程
1. 登录 → JWT
2. 用户管理 → admin 全权
3. 合同 CRUD + 状态流转
4. 附件上传/下载（类型+magic bytes 校验）
5. 异常：未登录重定向、无权限 403、文件拒绝、404

## 技术方案
- 后端：FastAPI + SQLAlchemy + SQLite
- 前端：Jinja2 SSR + 静态 CSS/JS
- 认证：JWT, 演示账号 admin/admin123, manager/manager123, viewer/viewer123
- Python 环境：~/外部需求/.conda/codingagent (Python 3.11)
- 部署：systemd + Nginx, 端口 8100
