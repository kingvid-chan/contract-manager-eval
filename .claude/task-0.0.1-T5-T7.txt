## T5: 前端模板 + 静态资源

静态资源路径格式：/projects/contract-manager-eval/static/{css,js}/*?v=0.0.1

### static/css/style.css
现代简洁的企业后台风格：
- 白色背景 #fff，深色文字 #1a1a2e
- 主色调 #2563eb（蓝色），危险 #dc2626（红），成功 #16a34a（绿）
- 导航栏：深色背景 #1e293b，白色文字，sticky top
- 表格：斑马纹，hover 高亮，圆角边框
- 表单：标签在上，input/select/textarea 全宽，focus 蓝色边框
- 按钮：.btn, .btn-primary, .btn-danger, .btn-sm
- 状态标签：.badge, .badge-draft, .badge-pending, .badge-signed, .badge-terminated, .badge-expired
- 响应式基础（max-width 容器）
- 登录页居中卡片样式

### static/js/app.js
纯 JavaScript（无框架）：
- API 请求封装：apiGet, apiPost, apiPut, apiDelete（自动带 Authorization Bearer token，根据响应状态处理 401 跳转登录）
- Token 管理：localStorage 存取
- 合同状态流转按钮交互
- 附件上传预览（文件名显示）
- 表单提交（阻止默认，JSON/FormData，显示错误/成功消息）
- 用户角色/状态下拉联动

### templates/
使用 Jinja2 模板语法。所有 URL 用 base_path 前缀。

#### templates/base.html
- HTML5 doctype, lang="zh-CN"
- viewport meta 标签
- 引入 static/css/style.css?v=0.0.1 和 static/js/app.js?v=0.0.1
- 导航栏：左侧"合同管理系统"，右侧根据 user 角色显示链接
  - 所有用户：合同列表
  - manager/admin：新建合同
  - admin：用户管理
  - 已登录：用户名 + 角色标签 + 退出按钮
- {% block content %}{% endblock %}
- 底部加载 app.js

#### templates/login.html
- 居中卡片（400px 宽）
- 标题"合同管理系统"
- 用户名输入框 + 密码输入框
- 登录按钮（蓝色主色调）

#### templates/contracts/list.html
- 搜索栏：状态下拉筛选（全部/草稿/待签/已签/终止/过期）+ 关键词搜索输入框
